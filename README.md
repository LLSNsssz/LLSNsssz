## 자기소개
다양한 산업 분야에서의 경험을 바탕으로 소프트웨어 개발에 도전한 주니어 개발자입니다. 프린터 용품 도매, 가구 재고관리, 반도체 후처리 분야에서 겪은 다양한 문제 상황을 해결하며 얻은 로직적 사고와 환경 적응력을 개발 업무에 적용하고 있습니다. Java와 Spring 프레임워크를 활용한 웹 애플리케이션 개발 과정에서 데이터베이스 설계, API 구현, 그리고 프론트엔드와의 협업을 통해 사용자 중심의 기능을 구현한 경험이 있습니다. 코드 리뷰를 통해 팀원들과 지식을 공유하고, 클린 코드 작성을 위해 지속적으로 리팩토링하며 성능 개선에 노력합니다. 현재는 테스트 주도 개발(TDD)과 마이크로서비스 아키텍처에 관심을 가지고 학습 중이며, 이를 실제 프로젝트에 적용하여 더 견고하고 확장 가능한 시스템을 구축하는 것이 목표입니다. 
## 🌱 배우고 있는 것들
- Java
- Spring Framework
- JPA (Java Persistence API)
